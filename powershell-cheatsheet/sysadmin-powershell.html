⚡ Base y Sintaxis PowerShell
💻
PowerShell
Shell orientada a objetos. Los cmdlets devuelven objetos, no texto. Encadena con | y filtra con Select-Object, Where-Object y ForEach-Object
💡
Conceptos fundamentales
Lo que necesitas saber antes de empezar
▼
Verb-NounTodos los cmdlets siguen este patron: Get-Process, Set-Item, New-ADUser
Pipeline |Pasa el objeto de salida al siguiente cmdlet como entrada
$variableVariables empiezan con $. Tipado dinamico: $x = 5 o $x = "texto"
Get-HelpDocumentacion integrada. Get-Help Get-Process -Examples muestra ejemplos reales
Get-MemberVer propiedades y metodos de cualquier objeto: Get-Process | Get-Member
Alias utilesls=Get-ChildItem, ps=Get-Process, cat=Get-Content, cd=Set-Location, echo=Write-Output
PS 5.1 vs 7.xPS 7.x es multiplataforma y mas rapido. Windows Server 2022 trae PS 5.1 por defecto

# Ver version instalada $PSVersionTable # Ayuda de cmdlet con ejemplos Get-Help Get-Process -Examples # Ver propiedades de un objeto Get-Process | Get-Member # Ejecutar como Admin desde PS normal Start-Process powershell -Verb RunAs # Cambiar politica de ejecucion Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Limpieza de cuentas inactivas
Deshabilita usuarios sin acceso en N dias
▼
💻 cleanup-inactivos.ps1
# ======================================== # DESHABILITAR USUARIOS INACTIVOS # ======================================== $DiasInactivo = 90 $OUExcluida = "OU=Cuentas_Servicio,DC=empresa,DC=local" $Log = "C:\Logs\inactivos_$(Get-Date -Format 'yyyyMMdd').log" $Corte = (Get-Date).AddDays(-$DiasInactivo) $inactivos = Get-ADUser -Filter "LastLogonDate -lt '$Corte' -and Enabled -eq 'True'" -Properties LastLogonDate,DistinguishedName | Where-Object { $_.DistinguishedName -notlike "*$OUExcluida*" } Write-Host "Inactivos encontrados: $($inactivos.Count)" foreach ($u in $inactivos) { try { Disable-ADAccount -Identity $u $msg = "OK - $($u.SamAccountName) - Ultimo: $($u.LastLogonDate)" Write-Host $msg -ForegroundColor Yellow } catch { $msg = "ERROR - $($u.SamAccountName): $($_.Exception.Message)" Write-Warning $msg } Add-Content $Log "$(Get-Date -Format 'HH:mm:ss') $msg" } Write-Host "Log: $Log" -ForegroundColor Cyan

Monitor de espacio en disco
Alerta cuando supera umbral en lista de servidores
▼
💻 disk-monitor.ps1
# ======================================== # MONITOR ESPACIO EN DISCO # ======================================== $Servers = @("SRV-DC01", "SRV-APP01", "SRV-FILE01") $Umbral = 85 $Result = @() foreach ($srv in $Servers) { try { $discos = Get-WmiObject -ComputerName $srv -Class Win32_LogicalDisk -Filter "DriveType=3" foreach ($d in $discos) { $pct = [math]::Round((($d.Size - $d.FreeSpace) / $d.Size) * 100, 1) $Result += [PSCustomObject]@{ Servidor=$srv; Disco=$d.DeviceID Total=[math]::Round($d.Size/1GB,1) Libre=[math]::Round($d.FreeSpace/1GB,1) Uso_Pct=$pct; Alerta=($pct -ge $Umbral) } if ($pct -ge $Umbral) { Write-Warning "ALERTA: $srv $($d.DeviceID) al ${pct}%" } } } catch { Write-Warning "Sin acceso a $srv" } } $Result | Format-Table -AutoSize $Result | Export-Csv "C:\Informes\discos_$(Get-Date -Format 'yyyyMMdd').csv" -NoTypeInformation

Health check del servidor
CPU, RAM, disco y servicios en un vistazo con colores
▼
💻 server-healthcheck.ps1
# ======================================== # HEALTH CHECK DEL SERVIDOR # ======================================== Clear-Host Write-Host "== HEALTH CHECK - $(hostname) - $(Get-Date -Format 'dd/MM HH:mm') ==" -ForegroundColor Cyan # Uptime $boot = (Get-CimInstance Win32_OperatingSystem).LastBootUpTime Write-Host "Uptime: $([math]::Round(((Get-Date)-$boot).TotalDays,1)) dias" # CPU $cpu = (Get-CimInstance Win32_Processor).LoadPercentage $c = if($cpu-gt85){"Red"}elseif($cpu-gt60){"Yellow"}else{"Green"} Write-Host "CPU: ${cpu}%" -ForegroundColor $c # RAM $os = Get-CimInstance Win32_OperatingSystem $ram = [math]::Round(($os.TotalVisibleMemorySize-$os.FreePhysicalMemory)/$os.TotalVisibleMemorySize*100,1) $c = if($ram-gt90){"Red"}elseif($ram-gt75){"Yellow"}else{"Green"} Write-Host "RAM: ${ram}% usada" -ForegroundColor $c # Discos Get-PSDrive -PSProvider FileSystem | ForEach-Object { $p = if($_.Used-gt0){[math]::Round($_.Used/($_.Used+$_.Free)*100,1)}else{0} $c = if($p-gt90){"Red"}elseif($p-gt80){"Yellow"}else{"Green"} Write-Host "Disco $($_.Name): ${p}% (Libre: $([math]::Round($_.Free/1GB,1))GB)" -ForegroundColor $c } # Servicios criticos foreach ($svc in @("Netlogon","DNS","W32Time","NTDS")) { try { $s= Get-Service $svc -EA Stop $c=if($s.Status-eq"Running"){"Green"}else{"Red"} Write-Host "$($s.Name): $($s.Status)" -ForegroundColor $c } catch { Write-Host "$svc: N/A" -ForegroundColor DarkGray } } Write-Host "==========================================" -ForegroundColor Cyan

Alta masiva de usuarios desde CSV
Crear N usuarios en AD de un fichero
▼

# ======================================== # ALTA MASIVA USUARIOS AD DESDE CSV # ======================================== $Csv = "C:\usuarios_nuevos.csv" $Pass = "Empresa2024!" $Log = "C:\Logs\alta_masiva_$(Get-Date -Format 'yyyyMMdd').log" Import-Csv $Csv | ForEach-Object { try { New-ADUser -Name $_.nombre -SamAccountName $_.sam -UserPrincipalName "$($_.sam)@empresa.local" -Path "$($_.ou),DC=empresa,DC=local" -Department $_.departamento -Title $_.titulo -AccountPassword (ConvertTo-SecureString $Pass -AsPlainText -Force) -Enabled $true -ChangePasswordAtLogon $true $msg = "OK - $($_.sam)" Write-Host $msg -ForegroundColor Green } catch { $msg = "ERROR - $($_.sam): $($_.Exception.Message)" Write-Warning $msg } Add-Content $Log "$(Get-Date -Format 'HH:mm:ss') $msg" }
